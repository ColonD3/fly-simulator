# Elytra Course

A browser-based 3D infinite terrain flight simulator with spawning hoops and score tracking.

## Run locally

From this project folder:

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173`.

## Controls

- `W/S`: Pitch up/down
- `A/D`: Roll
- `Q/E`: Yaw
- `Shift`: Boost
